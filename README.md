# Service Port Dashboard

本项目提供一个本地服务面板，用于查看监听端口、容器/进程信息，以及读取并切换部分本机模型配置。

## Security Notice

本项目的模型配置页面会直接展示从本机配置文件读取到的 API key。

这意味着：

- 如果你运行这个应用，且该应用可被他人访问，那么访问者可以直接看到你的 API key。
- 不要将本项目部署到公网可访问环境。
- 不要暴露给不受信任的局域网、反向代理、内网穿透或共享主机环境。
- 仅建议在你本人可控的本机环境中使用。

如果你不能接受这一行为，请自行修改代码，将 API key 改为脱敏显示或移除展示逻辑。
